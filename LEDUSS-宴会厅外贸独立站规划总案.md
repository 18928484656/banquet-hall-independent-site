## 8. AIO / GEO / AI 搜索优化策略

AIO = AI Overview Optimization，GEO = Generative Engine Optimization。目标是让 Google AI Overview、ChatGPT、Perplexity、Gemini 更容易理解并引用 LEDUSS。

### 必做内容结构

| 模块 | 做法 |
|---|---|
| 清晰实体 | 每页都明确写：LEDUSS is a banquet hall design and construction solution provider based in Foshan, China. |
| 结构化定义 | 服务页开头用 40-60 字解释该服务是什么 |
| FAQ | 每个核心服务页放 6-8 个真实问题 |
| How-it-works | 用步骤说明从咨询到交付的流程 |
| Proof | 引用真实数据：1000+ projects、430,000+ m²、12+ years、Oscar case |
| Comparison | 写清楚 Design-only vs Design-build vs Turnkey EPC 的区别 |
| Schema | Organization、LocalBusiness、Service、FAQPage、BreadcrumbList、Project/CaseStudy |
| 多格式内容 | 页面正文 + FAQ + 表格 + 案例 + 图片 ALT + 视频 transcript |
